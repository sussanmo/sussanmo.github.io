# suadhm.github.io

Academic webpage host repository.

## Quick start

1. Edit `index.html` with your name, affiliation, bio, research, publications, and contact links.
2. Update styles in `styles.css` if you want to change spacing, fonts, or colors.
3. Commit and push to the `main` branch.

## Publish with GitHub Pages

For a repository named `suadhm.github.io`, GitHub serves the site automatically from the root of the default branch.

- Site URL: `https://suadhm.github.io`
- Main page file: `index.html`

If Pages is not already enabled:

1. Open repository **Settings** on GitHub.
2. Go to **Pages**.
3. Under **Build and deployment**, choose:
	- **Source**: Deploy from a branch
	- **Branch**: `main` / `/ (root)`
4. Save and wait 1–2 minutes.
